# deployment-tiered-healing 规范

## 目的

定义 Deployment 在故障处置中的分层自愈语义：L1 执行最小影响动作，L1 失败后进入 L2 健康版本回滚，无法证明安全或恢复失败时降级为 L3 人工介入，并对整个升级、回滚与恢复过程输出可审计证据。

## 需求

### 需求:Deployment L1 最小影响动作

系统必须将 Deployment L1 定义为最小影响自动处置动作，并在动作前完成快照校验。

#### 场景: L1 动作成功

- **当** Deployment L1 动作执行成功
- **那么** 系统必须结束自动处置并进入持续观测

### 需求:Deployment L2 健康版本回滚

系统必须在 Deployment L1 失败后进入 L2 候选判定，并仅允许回滚到满足健康、依赖与风险门禁条件的版本；若任一条件无法满足，系统必须降级为 L3 人工介入。

#### 场景: L1 失败后进入 L2 并选择最近健康版本

- **当** Deployment L1 动作失败且允许继续自动处置
- **那么** 系统必须列出可用 revision，选择最近的健康候选，并在执行前记录候选、门禁决策和依赖校验结果

#### 场景: L2 候选不足降级

- **当** Deployment 不存在满足条件的健康候选
- **那么** 系统必须降级为 L3 人工介入，并输出 no-healthy-candidate 证据和下一步建议

#### 场景: L2 候选依赖校验失败降级

- **当** Deployment 的健康候选缺少所需依赖或依赖校验失败
- **那么** 系统必须降级为 L3 人工介入，并输出 dependency-validation-failed 证据

#### 场景: L2 风险门禁阻断

- **当** Deployment 的 L2 回滚窗口失败率或降级率超过策略阈值
- **那么** 系统必须阻断 L2 自动写操作，并输出门禁原因码、阈值快照和人工介入建议

#### 场景: L2 回滚成功

- **当** Deployment 存在满足条件的健康候选且回滚执行成功
- **那么** 系统必须记录 rollback-succeeded 结果、更新最近健康 revision，并结束自动处置

### 需求:Deployment L2 失败后快照恢复保护

系统必须在 Deployment L2 健康版本回滚执行失败时，立即尝试恢复 L1 前快照，并输出恢复结果与人工介入建议。

#### 场景: L2 回滚失败后恢复快照成功

- **当** Deployment 已进入 L2 且回滚到健康 revision 失败，但 L1 前快照恢复成功
- **那么** 系统必须记录 rollback failed 与 snapshot restored 的联合证据，并阻断后续自动写操作

#### 场景: L2 回滚失败且恢复快照失败

- **当** Deployment 已进入 L2 且回滚到健康 revision 失败，同时 L1 前快照恢复也失败
- **那么** 系统必须输出恢复失败原因、进入人工介入语义，并给出可执行的下一步建议

### 需求:Deployment L3 人工介入建议

系统必须在无法安全自动处置时进入 L3，并输出包含候选缺失、门禁阻断、回滚失败与恢复结果的结构化人工介入建议。

#### 场景: 自动处置不可证明安全

- **当** Deployment 路径命中证据不足或安全门禁阻断
- **那么** 系统必须进入 L3，并给出包含原因码、候选状态和建议动作的下一步建议

#### 场景: L2 失败后输出恢复导向建议

- **当** Deployment 的 L2 回滚失败
- **那么** 系统必须在人工介入建议中附带 snapshot restore 结果、最近候选 revision 和建议检查项

### 需求:Deployment 分层审计可解释

系统必须为 Deployment 每次阶段迁移记录触发条件、失败证据、执行结果与下一步建议；并必须附带门禁判定结果、原因码与阈值快照。

#### 场景: 阶段迁移审计输出

- **当** Deployment 发生阶段迁移
- **那么** 系统必须输出可检索的阶段审计记录

#### 场景: 审计附带门禁原因码

- **当** Deployment 阶段迁移命中门禁阻断或降级
- **那么** 系统必须输出门禁原因码与阈值快照

### 需求:Deployment 分层门禁证据对齐

系统必须将 Deployment 三阶段迁移事件与门禁判定结果关联输出，确保单次处置可追溯到对应门禁决策。

#### 场景: 阶段迁移关联门禁决策

- **当** Deployment 发生阶段迁移并触发门禁评估
- **那么** 系统必须输出阶段迁移与门禁判定的关联证据
