## ADDED Requirements

### 需求:执行链路去硬编码

系统必须禁止在运行时执行链路中硬编码业务阈值与时间窗口，必须由 CRD 字段或配置项驱动。

#### 场景: 门禁参数读取

- **当** 编排链路执行安全门禁判定
- **那么** 系统必须从请求配置读取幂等窗口、限频窗口、爆炸半径与熔断阈值

### 需求:运行态事实输入

系统必须基于运行态事实数据完成门禁判定，禁止使用固定示例值替代真实输入。

#### 场景: 爆炸半径判定

- **当** 系统评估一次自动处置的影响范围
- **那么** 系统必须使用实时受影响 Pod 数与集群 Pod 总数计算判定结果

### 需求:生产可采集观测输出

系统必须提供 Prometheus 指标与 K8s Event 输出，并与审计记录通过关联键串联。

#### 场景: 自愈请求完成

- **当** 一次自愈链路进入完成或阻断状态
- **那么** 系统必须输出可采集指标、可检索事件与可追溯审计记录
