## 目的

定义生产环境中 StatefulSet Phase 2 的灰度发布、指标门禁与参数配置要求，确保可控启用、可观测回退与最小风险上线。
## 需求
### 需求:Phase 2 灰度发布控制

系统必须支持 StatefulSet Phase 2 的灰度发布控制，至少包含命名空间白名单、对象级审批与快速回滚开关。

#### 场景: 灰度范围受控

- **当** 运维仅对部分命名空间启用 Phase 2
- **那么** 系统必须确保非白名单命名空间继续保持只读策略

### 需求:Phase 2 指标门禁

系统必须为 StatefulSet Phase 2 输出误动作率、回退率、冻结触发率等关键指标，并作为发布阻断依据。

#### 场景: 指标越线回滚

- **当** 任一 Phase 2 关键指标超过阈值
- **那么** 系统必须触发自动降级或提示立即回滚到只读模式

### 需求:执行链路去硬编码
系统必须将 StatefulSet L2 候选窗口、冻结时长、灰度阈值与降级条件全部配置化，禁止硬编码策略值；同时必须将快照保留窗口、恢复超时、每对象快照上限等参数配置化。

#### 场景: L2 参数读取
- **当** 系统执行 StatefulSet L2 判定与执行
- **那么** 必须从声明式配置读取全部 L2 控制参数

#### 场景: 快照策略参数读取
- **当** 系统执行快照创建或恢复
- **那么** 必须从声明式配置读取快照策略参数并应用边界约束

### 需求:StatefulSet L2 灰度发布控制
系统必须支持 StatefulSet L2 的灰度发布控制，至少包含命名空间白名单、审批开关、快速回退开关与冻结阈值。

#### 场景: L2 灰度范围受控
- **当** 仅对白名单命名空间开启 StatefulSet L2
- **那么** 非白名单命名空间必须继续保持 L1+L3 路径

### 需求:StatefulSet L2 发布阻断指标
系统必须输出 StatefulSet L2 成功率、失败回退率、冻结触发率与降级率，并作为发布阻断依据。

#### 场景: 指标越线阻断
- **当** 任一 L2 关键指标超过阈值
- **那么** 系统必须自动降级并提示回退到保守模式

### 需求:快照容量与时延发布门禁
系统必须输出快照创建成功率、恢复成功率、恢复时延与对象级快照容量占用，并将其作为发布阻断指标。

#### 场景: 快照指标越线阻断
- **当** 快照失败率或恢复时延超过阈值
- **那么** 系统必须阻断继续灰度并提示回退到保守模式
