## ADDED Requirements

### 需求:门禁结果映射事故响应等级
系统必须将发布门禁结果映射为事故响应等级，并输出对应值班动作建议；其中 `block` 必须触发最高优先级响应路径。

#### 场景: block 触发最高优先级响应
- **当** 发布门禁结果为 `block`
- **那么** 系统必须输出最高优先级响应等级并要求人工确认后方可继续

### 需求:降级路径恢复条件约束
系统必须在 `degrade` 状态下输出明确恢复条件，未满足恢复条件前禁止自动回到放量路径。

#### 场景: degrade 状态未满足恢复条件
- **当** 系统处于 `degrade` 且恢复条件未满足
- **那么** 系统必须保持保守路径并持续输出恢复差距信息
