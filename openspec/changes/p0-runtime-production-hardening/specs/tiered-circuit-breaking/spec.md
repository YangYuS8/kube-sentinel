## MODIFIED Requirements

### 需求:分级治理优先级

系统必须遵循双层熔断优先级，并且熔断阈值、冷却时间必须由请求配置驱动。

#### 场景: 配置阈值生效

- **当** 请求设置对象级与域级熔断阈值
- **那么** 系统必须按请求阈值判断是否开启对应层级熔断

### 需求:熔断触发可视化

系统必须在熔断触发时输出熔断层级、触发阈值、当前计数与预计恢复时间。

#### 场景: 域级熔断触发

- **当** 失败密度达到域级阈值
- **那么** 系统必须记录域级阈值、当前计数与恢复时间并输出可检索事件
