## ADDED Requirements

### 需求:预生产验证通过前禁止生产灰度
运行时闭环验证必须将预生产验证结果作为生产灰度前置条件，未通过时禁止放量并要求先完成修复验证。

#### 场景: 预生产验证失败
- **当** 闭环验证输出失败或证据不完整
- **那么** 系统必须阻断生产灰度并输出修复建议

#### 场景: 预生产验证通过
- **当** 闭环验证输出通过且证据完整
- **那么** 系统必须允许进入生产灰度放量判定

### 需求:回滚演练达标阈值
运行时闭环验证必须定义回滚演练达标阈值，未达阈值时必须阻断 go-live。

#### 场景: 回滚演练未达标
- **当** 回滚演练成功率或恢复时延未达阈值
- **那么** 系统必须输出 block 并要求先完成补充演练

#### 场景: 回滚演练达标
- **当** 回滚演练满足成功率与恢复时延阈值
- **那么** 系统必须允许该项通过并进入下一闸门
