## ADDED Requirements

### 需求:Deployment 分层发布门禁
系统必须为 Deployment 分层路径输出 L1 成功率、L2 回滚成功率、L3 降级率与分层阻断率，并将其作为发布阻断指标。

#### 场景: Deployment 分层指标越线阻断
- **当** Deployment 分层关键指标超过阈值
- **那么** 系统必须阻断继续灰度并提示回退到保守模式

## MODIFIED Requirements

### 需求:执行链路去硬编码
系统必须将 StatefulSet L2 候选窗口、冻结时长、灰度阈值与降级条件全部配置化，禁止硬编码策略值；同时必须将快照保留窗口、恢复超时、每对象快照上限等参数配置化；Deployment 分层升级阈值与阻断条件也必须配置化。

#### 场景: L2 参数读取
- **当** 系统执行 StatefulSet L2 判定与执行
- **那么** 必须从声明式配置读取全部 L2 控制参数

#### 场景: 快照策略参数读取
- **当** 系统执行快照创建或恢复
- **那么** 必须从声明式配置读取快照策略参数并应用边界约束

#### 场景: Deployment 分层参数读取
- **当** 系统执行 Deployment L1/L2/L3 分层迁移判定
- **那么** 必须从声明式配置读取升级与阻断参数
