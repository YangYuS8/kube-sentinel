## ADDED Requirements

### 需求:生产门禁判定报告输出

系统必须在每次灰度窗口结束后输出门禁判定报告，包含判定结果、越线项、回退动作与恢复条件。

#### 场景: 灰度窗口结束生成报告

- **当** 灰度观测窗口结束
- **那么** 系统必须生成可检索的门禁判定报告

## MODIFIED Requirements

### 需求:Deployment 分层发布门禁

系统必须为 Deployment 分层路径输出 L1 成功率、L2 回滚成功率、L3 降级率与分层阻断率，并将其作为发布阻断指标；当任一指标越线时必须输出统一原因码并执行保守降级动作。

#### 场景: Deployment 分层指标越线阻断

- **当** Deployment 分层关键指标超过阈值
- **那么** 系统必须阻断继续灰度并提示回退到保守模式

#### 场景: Deployment 指标越线触发统一原因码

- **当** Deployment 发布门禁因任一指标越线被阻断
- **那么** 系统必须输出统一原因码、阈值快照与恢复建议
