## ADDED Requirements

### 需求:值班交接强制契约

系统必须要求 pilot 切流前完成值班交接字段校验，至少包含 `handoffOwner`、`approvalLevel`、`traceKey`、`rollbackCommandRef`、`handoffTimestamp`。

#### 场景: 交接字段完整

- **当** 值班交接字段均完整且审批级别匹配
- **那么** 系统必须允许进入 pilot 观察阶段

#### 场景: 交接字段缺失

- **当** 任一交接字段缺失
- **那么** 系统必须阻断切流并输出缺失字段列表

### 需求:冻结窗口内人工放量限制

系统必须在冻结窗口内禁止人工覆盖放量，仅允许只读评估与告警。

#### 场景: 冻结窗口命中

- **当** 当前时间命中冻结窗口且请求人工覆盖
- **那么** 系统必须拒绝覆盖并输出 `freeze_window_override_blocked`

#### 场景: 冻结窗口外受控覆盖

- **当** 当前时间不在冻结窗口且审批满足
- **那么** 系统必须允许受控覆盖并记录审计事件
