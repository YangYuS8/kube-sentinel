## ADDED Requirements

### 需求:生产门禁输出值班摘要
生产门禁必须输出值班可执行摘要，至少包含门禁结论、值班级别、审批要求与建议回滚路径。

#### 场景: 生产门禁触发阻断
- **当** 生产门禁判定为 `block`
- **那么** 系统必须输出值班摘要并要求审批后再继续

### 需求:降级状态持续观测模板
生产门禁必须在 `degrade` 状态输出持续观测模板，至少包含观测窗口、关键指标和恢复阈值。

#### 场景: 持续 degrade 状态
- **当** 门禁连续窗口保持 `degrade`
- **那么** 系统必须输出持续观测模板并禁止无条件放量
