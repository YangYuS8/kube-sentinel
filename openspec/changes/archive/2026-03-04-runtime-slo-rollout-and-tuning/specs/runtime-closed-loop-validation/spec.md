## ADDED Requirements

### 需求:灰度放量与回退演练断言

系统必须在闭环演练中验证灰度放量步骤、回退触发条件与回退执行证据。

#### 场景: 灰度阶段触发回退

- **当** 演练注入预算越线导致回退条件命中
- **那么** 系统必须执行回退并输出阶段切换证据

### 需求:阈值调优闭环断言

系统必须验证阈值调优流程包含审批、观察窗口和回滚路径，缺失任一环节即判定失败。

#### 场景: 调优流程缺少审批

- **当** 演练执行阈值调优但缺少审批记录
- **那么** 系统必须判定演练失败并阻断调优生效

### 需求:复盘动作可审计断言

系统必须验证复盘输出至少包含越线原因、处置动作、阈值调整结论与后续观察计划。

#### 场景: 复盘字段缺失

- **当** 演练输出复盘报告但缺少关键字段
- **那么** 系统必须判定闭环验收失败
