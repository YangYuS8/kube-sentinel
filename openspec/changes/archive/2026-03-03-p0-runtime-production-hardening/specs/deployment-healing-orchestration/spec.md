## MODIFIED Requirements

### 需求:编排链路幂等执行

系统必须保证同一告警在重复触发时不会导致重复副作用，必须以目标对象当前状态作为判定依据，并使用请求声明的幂等窗口执行重复抑制。

#### 场景: 自定义幂等窗口生效

- **当** 请求显式配置 `idempotencyWindowMinutes`
- **那么** 系统必须按该窗口判定重复事件并禁止重复写操作

### 需求:v1alpha1 Deployment 自愈编排范围

系统必须在 v1alpha1 阶段仅对 Deployment 执行自动自愈动作，并且门禁判定必须基于运行态事实输入。

#### 场景: 运行态门禁输入

- **当** 编排链路执行维护窗口、速率限制与爆炸半径判定
- **那么** 系统必须使用实时动作计数与规模数据，不得使用固定示例值
