## ADDED Requirements

### 需求:Deployment L1 首发自动处置闭环

系统必须在首发版本中为 Deployment 提供单一 L1 自动动作闭环，并且该动作必须只在门禁通过和快照创建成功后执行。

#### 场景: Deployment L1 执行成功

- **当** Deployment 请求通过所有前置门禁且写前快照创建成功
- **那么** 系统必须执行首发定义的唯一 L1 动作，并记录成功阶段与证据

### 需求:Deployment 首发版本禁止自动升级

系统必须在首发版本中禁止将 Deployment 从 L1 自动升级到更激进的自动动作；L1 失败时必须输出阻断、降级或人工介入建议，而不是继续执行 L2/L3。

#### 场景: Deployment L1 失败

- **当** 首发版本中的 Deployment L1 动作失败或任一前置条件失败
- **那么** 系统必须停止自动升级并输出可解释的失败原因与下一步建议
